SVG

SVG står for Scalable vector Graphics . Det vil si at du kan få det så stort du vil uten at det går utover kvaliteten. Det blir definert som lesbar kode XML og er et vektorbasert bildeformat. Det kan manipuleres manuelt, CSS eller JavaScript

Når brukes SVG på nettet ?
Det brukes i Logoer, Ikoner, Animasjoner, Ilustrasjoner og Skjema/grafer.

SVG har så komplekse og enkle farger men SVG er ikke en erstatning av JPG  

Om det er så enkelt, hvordan brukes det?
  - Du kan skrive det rett inn i HTML.
- Du kan skrive det via andre elementer.
- Du kan bruke image taggen.
- Du kan bruke objekttaggen.
- Eller du kan bruke Iframe- taggen.
